The diffrences between web1, web2 and web3 are:

Web 1.0 - The Web

  • Static pages - Read only web
  • used for information sharing -> connect information
  • Users can not interact or comment on the page
  • Content is served by the companies that people consume

Web 2.0 - the Social Web

  • Users can interact with the web page - Read and write web
  • Used for interaction - connect people
  • Highlight user-generated content, usability and interoperability for end users
  • Interaction and collaboration with each other is allowed by social media dialogue as creator of user-generated content in a virtual community

Web 3.0 - The Semantic Web

  • Internet of things
  • Used for Immersion - connect knowledge
  • People build applications that people can interact with, while the companies build the platforms that let them publish services
  • Allow for "things" or inanimate objects to be wired with proccessors & connected by sensors
  • Big data, AI, Cloud Computing
