The diffrences between web1, web2 and web3 are:

Web 1.0 - The Web

  • Static pages - Read only web
  • Used for information sharing -> connect information
  • Users can not interact or comment on the page
  • Content is served by the companies that people consume

Web 2.0 - The Social Web

  • Dynamic pages - Read and write web
  • Used for interaction - connect people
  • Highlight user-generated content, usability and interoperability for end users
  • Social netrworks, RSS, Media sharing, Blogs
  • Peer to Peer

Web 3.0 - The Semantic Web

  • Internet of things - linked data sets
  • Used for Immersion - connect knowledge
  • People build applications that people can interact with, while the companies build the platforms that let them publish services
  • Allow for "things" or inanimate objects to be wired with proccessors & connected by sensors
  • Big data, AI, Cloud Computing

Use Case:


If we try searching for free gluten recipes for example, we will get different results in each of these web versions.

Web 1.0: The results are of static pages of probably big companies and famous chefs. What was published by them is what we can consume.
Web 2.0: In this case, the results will also include posts and comments made by people over the internet. This might help with deciding if the recipe is good or not.
Web 3.0: This time, the results, as well as information and comments, will be linked to other features. Maybe buy some of the ingredients (Shopping bag).
We also might get suggetsions following our search for further things as the web trying to learn what we want. for example: sugar free recpies because it all might be semantically related to diet
